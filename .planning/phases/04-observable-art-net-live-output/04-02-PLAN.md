## Phase Goal

**As an** Art-Net operator, **I want to** drive a small lighting rig with correct Art-Net 4 frames computed from the deterministic playback engine and independently verify protocol, target, and timing health, **so that** I can run reliable live output without needing the desktop UI.

> Derived from the ROADMAP.md Phase 4 `**Goal:**` line (prose form). All three story slots are present in the goal; this is a restatement. Run `/gsd mvp-phase 4` if the ROADMAP goal line itself should be normalized.

<objective>
Deliver the two configuration primitives the worker needs: interface selection with pinned-interface loss detection (ARTN-01, D-05) and the unicast target model with fan-out and per-target enable/disable (ARTN-02, D-07/D-08/D-12). After this plan an operator's selected NIC and configured targets are represented and validated, so Plan 03's worker has a real interface to bind and real targets to fan out to.

Purpose: Correct, safe network targeting is a prerequisite for observable output. Pinning by index and never auto-switching (D-05), unicast-only (D-07), and independent enable/disable (D-12) are safety-relevant behaviors that must be modeled before any send happens.
Output: `interfacemgr.go` + `target.go` in `internal/artnet`.
</objective>

Create `internal/artnet/interfacemgr.go` (package `artnet`). Implement `ListCandidateInterfaces() ([]InterfaceInfo, error)` per RESEARCH Pattern 2: call `net.Interfaces()`, wrap enumeration failure as `GOLC_ARTNET_INTERFACE_ENUM_FAILED`, and for each interface capture Index, Name, `Flags&net.FlagUp != 0`, and `Addrs()`. `InterfaceInfo` is `{Index int, Name string, Up bool, Addrs []net.Addr}`.

Implement `InterfaceManager` holding the pinned `Index` (persist by index; keep Name for display only — Pitfall 4). Add `pollInterfaceLoss(ctx context.Context)` on its own 1Hz ticker (independent of the 40Hz send path), mirroring engine.go's `context.WithCancel`/goroutine/`select { case <-ctx.Done(); case <-ticker.C }` lifecycle: each tick, re-resolve `net.InterfaceByIndex(pinnedIndex)` and, on error or down state, call `markLost()`, which flips a readable degraded/error status (`GOLC_ARTNET_INTERFACE_LOST`). Never call any code path that selects a different interface index automatically (D-05) — loss is terminal-until-reconfigured, not auto-recovered onto another NIC.

Provide a `LocalIP()`/bind-address accessor returning the pinned interface's local unicast IP so the worker (Plan 03) binds `net.ListenUDP`/`net.DialUDP` to that local address rather than 0.0.0.0 or a Linux-only device-bind socket option (Pitfall 5). Expose a status accessor safe to read from a concurrent goroutine (atomic or mutex-guarded).

Create `internal/artnet/interfacemgr_test.go`: assert ListCandidateInterfaces returns the loopback interface on the test host; assert markLost transitions status to lost; assert an InterfaceManager pinned to a bogus index is reported lost by a single poll iteration (call the poll body directly rather than sleeping on the ticker).
