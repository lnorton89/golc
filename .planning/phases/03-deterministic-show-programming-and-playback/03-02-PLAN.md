## Phase Goal

**As a** show author, **I want to** capture reusable themes and presets from what I have programmed, **so that** I can reuse looks across scenes without re-authoring them each time.

This slice (03-02) turns transient programmer state (03-01) into durable, named, reusable programming objects — the building blocks scene layers reference.

<objective>
Deliver reusable color themes and intensity/color/position/beam presets (PROG-04), recorded from the 03-01 programmer buffer, persisted as new `Themes` and `Presets` fields on `show.State`, and driven by `theme create` / `preset record` command routes.

Purpose: Themes and presets are the first reusable programming objects; chases (03-03) and scene color-theme/base-look layers (03-04) reference them.
Output: `internal/programming/theme.go`, `internal/programming/preset.go`, extended `show.State`, `theme`/`preset` command routes.
</objective>
